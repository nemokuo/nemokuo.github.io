What is Riso?印刷特性

Riso是一台透過單色疊印的數位印刷機，使用無毒環保的大豆油墨印製，印刷原理類似於傳統網印技法，除了特殊的復古網點效果，完成品總有不可預期的錯位變化。由於紙張皆選用非塗佈的紙張印製，成品看起來總是極具手感的氛圍，使得每張作品呈現不盡相同。

    ///完成品特性注意///
  • 錯位：因逐次疊印，每色的印刷位置不易對位精準，完成品易產生錯位位移。
  • 不易乾燥：Riso主要依紙張吸附油墨，久置乾燥，但仍無法完全至乾燥。
  • 墨跡：在濃度調整較高時易有不均勻墨跡產生（每色油墨表現不同）。

以上均屬Riso印刷正常表現，請於印製前務必確認無疑慮。

檔案交付>>>類似網版印刷原理，印前需將完成設計稿製作分色檔，並調整為灰階濃度（請參閱稿件設定圖解），檔案以AI/PSD/PDF交稿。RISO印刷使用紙張最大尺寸為A3（297x420mm），滿版印刷時四邊須留10mm離邊，最大可印面積為420x277mm，其他特殊尺寸請另行聯繫詢問。

File submission>>>Just as with screen printing, your artwork should be setup as separate greyscale PDF/AI/PSD files, one for each color layer. Please note that the Riso cannot print full bleed, and large areas of block color won't print out evenly, and will also cause print marks. There is a 10mm border on prints. Maximum Size: Ledger (420 x297mm ) - A3 Printable area aprox.: 400 x 277mm
